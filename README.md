explr
=====

A Shiny App For Viewing a dataset in up to 5 dimensions at once

Install from GitHub:

  If you don't have devtoolsi installed:
  
  > install.packages('devtools')
  > library(devtools)
  
  Then:
  
  > install_github('explr', 'johncollins')
  > library(explr)
  
  And explore away to your hearts content:
  
  > library(MASS)
  > explr(Aids2)
